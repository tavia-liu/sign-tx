# Signed Ethereum Transaction (Sepolia)

This project demonstrates how to send a **signed Ethereum transaction**
on the **Sepolia testnet** using the **Alchemy SDK** and **Ethers.js**.

## Features
- Uses Alchemy SDK to connect to Sepolia
- Signs transactions locally with a private key
- Sends EIP-1559 (type 2) transactions
- Uses environment variables for secrets
